== Description ==

Exclusive is a versatile and responsive WordPress theme suitable for blogs, portfolios, and various types of websites. It features a clean and modern design, customizable layout options, and built-in support for popular plugins.

== Installation ==

1. Download the Exclusive theme zip file from your account on our website.
2. Log in to your WordPress admin panel.
3. Go to Appearance > Themes.
4. Click on the "Add New" button, then click on the "Upload Theme" button.
5. Choose the Exclusive theme zip file and click "Install Now".
6. After installation, click on the "Activate" button to activate the theme.
7. Customize the theme settings by going to Appearance > Customize.
